# GIthubFollowers

A modified version of [Take home project](https://www.youtube.com/watch?v=JzngncpZLuw&t=45660s&pp=ygUUc2VhbiBhbGxlbiB0YWtlIGhvbWU%3D) by Sean Allen. Using UIKit, MVVM and Combine.

## Features:
- UIKit
- Programmatic UI
- MVVM pattern
- Combine
- Pagination
- Caching image
- UserDefaults
- Delegates/Protocols
- Light/Dark mode
- Unit test
